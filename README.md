# LLM 기반 금융상품 추천 에이전트 (Colab + RAG)

본 프로젝트는 **공개 금융 데이터셋(GitHub 기반 Yahoo Finance 샘플)**을 이용해  
사용자의 투자 성향·기간·관심 섹터 등 자연어 입력을 분석하고  
관련 종목을 자동으로 검색·선별해 추천하는 **금융상품 추천 에이전트**를 구현함.

LLM과 벡터 검색(RAG) 구조를 결합하여 다음과 같은 전체 흐름을 자동화함:

- 금융 데이터셋 로딩 및 종목 정보 정제  
- 종목 설명을 임베딩하여 FAISS 벡터 인덱스 구축  
- 사용자 프로필 기반 질의 임베딩  
- 질의와 가장 유사한 종목 후보 탐색  
- 후보 종목 분석 및 이유·리스크를 포함한 추천 문장 생성  

---

## 1. 프로젝트 개요

일반적인 투자자들은 종목명이나 시가총액 정도만 알고,  
자신의 상황에 맞는 투자 대상을 직접 고르는 데 어려움을 겪는 경우가 많음.

본 프로젝트는 다음 문제를 해결하는 것을 목표로 함:

- **자연어 기반 “투자 조건” 해석**
- **조건과 유사한 종목 자동 탐색**
- **후보 중 적합한 종목 선별**
- **추천 이유·리스크까지 자연어로 정리**

---

## 2. 주요 기능

### ● 자연어 기반 종목 검색
사용자의 투자 성향, 기간, 금액, 선호 섹터 등을 문장 형태로 입력받아  
관련도 높은 종목을 벡터 검색 기반으로 자동 탐색함.

### ● RAG 기반 종목 추천
채택된 Top-K 종목들을 LLM이 분석하여  
“이 사용자의 조건에 맞는지”를 이유 기반으로 선별하여 텍스트로 설명함.

### ● 금융 데이터셋 자동 처리
GitHub의 공개 CSV(Yahoo Finance 샘플 데이터)를 직접 다운로드하여  
종목명, 티커, 시가총액, 배당수익률, PER, 기업 요약 등 텍스트 정보를 자동 정제함.

### ● 임베딩 + FAISS 인덱스 구축
각 종목을 설명형 문장으로 변환한 뒤 임베딩해 FAISS에 저장.  
사용자 질문에 대한 임베딩과 비교하여 관련 종목을 효율적으로 조회함.

### ● Colab 단일 파일 구조
서버 실행 없이 Colab에서 한 파일로 모두 구동되며  
코드 실행 → 질문 입력만으로 결과 확인 가능함.

## 3. 시스템 흐름 (Workflow)

[User Profile & Question]  
→ [텍스트 결합 후 임베딩 생성]  
→ [FAISS 벡터 검색으로 유사 종목 Top-K 조회]  
→ [후보 종목들의 설명 텍스트 생성]  
→ [LLM 분석 및 추천 종목 결정]  
→ [추천 이유/리스크 자연어 응답]
